# Contributing

Thanks for contributing to **Minecraft Python Failover Proxy**.

## Quick workflow

1. Fork the repository.
2. Create a feature/fix branch.
3. Make your changes.
4. Test locally.
5. Open a pull request.

## Coding style

- Prefer Python standard library solutions.
- Keep configuration simple and explicit.
- Avoid unnecessary external dependencies.
- Keep `README.md` and `README.de.md` in sync when behavior changes.
